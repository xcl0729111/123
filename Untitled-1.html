应用设计说明

用户群体：双向情感障碍患者及其支持者，包括医生、家人等

应用目标：提供全面的情绪记录、VR绘画治疗、社区支持和个性化推荐功能，帮助患者更好地管理情绪波动

核心功能模块：

  • 用户调研与个性化推荐 - 基于MBTI和情绪数据的个性化疗愈方案
  • VR绘画治疗 - 沉浸式创作体验，情绪表达与调节
  • 社交功能与社区 - 提供安全的交流与支持环境
  • 用户情绪与疗愈记录 - 跟踪情绪变化，记录疗愈进展

用户场景与功能分析

核心用户场景

  1. 情绪低落时的自我管理

    用户在感到情绪低落时，可以打开APP记录自己的情绪，系统根据MBTI个性分析和历史数据，推荐个性化的干预措施（如VR绘画、冥想）

  2. 预约和进行专业辅导

    用户可以在APP中预约专业医生进行远程辅导，并在预约时间到达时接收提醒，直接在应用内进行视频或文字沟通

  3. 社区支持与情感分享

    用户可以在社区中分享自己的情绪管理经验、VR绘画作品，与同样经历情绪障碍的人建立联系，获得理解和支持

  4. 日常情绪监测与干预

    用户每天记录自己的情绪和触发因素，系统会生成情绪分析报告，并提供针对性建议，帮助用户认识自己的情绪模式

无缝医疗连接

与专业医疗资源无缝对接，支持在线预约、远程咨询和数据分享，方便医生掌握患者情况并提供精准干预

React Native

用于构建跨平台移动应用，确保iOS和Android用户体验一致性

为VR绘画功能提供3D渲染和交互支持

TailwindCSS

用于构建一致、可扩展的用户界面

构建高性能API服务和实时通信功能

存储用户数据、情绪记录和社区内容

Redis

缓存和实时数据处理

AI和数据分析

个性化推荐系统

基于协同过滤和内容过滤的混合推荐算法，根据用户情绪数据和反馈不断优化推荐内容

情绪分析引擎

基于NLP和情感分析技术，分析用户文本输入和互动行为，识别情绪状态

VR绘画分析

结合计算机视觉和情感计算，分析用户绘画的线条、色彩和构图，推断情绪状态

预测模型

通过机器学习建立情绪波动预测模型，帮助用户提前干预可能的情绪变化
